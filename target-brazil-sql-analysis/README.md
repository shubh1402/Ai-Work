# Target Brazil SQL Analysis

## Overview
This project analyzes Target Brazil’s e-commerce dataset using SQL to uncover insights related to customer behavior, order trends, delivery performance, payment methods, and logistics efficiency.

## Business Problem
As a data analyst at Target, the objective was to analyze e-commerce operations in Brazil and provide actionable business insights using SQL-based analytics.

## Key Analysis Areas
- Exploratory Data Analysis
- Order Trend Analysis
- Customer Distribution
- Delivery Performance
- Freight & Payment Analysis
- Seasonal Trends
- Business Recommendations

## Tech Stack
- SQL
- MySQL
- Data Analytics
- Business Intelligence
